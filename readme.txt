Overall 24 options can be configured for branding and customizing a WordPress installation.

== Description ==

Totally brand your WP backend with 24 options!

Features:

*   Change admin bar logo.
*   Change login screen logo.
*   Style login screen.
*   Change footer text.
*   Configure dashboard.
*   Add widgets like RSS to dashboard. 
*   Add style and script to admin screens.
*   Remove metaboxes.
*   Remove admin menu items.
*   Remove widgets.

== Installation ==

1. Install The Blackest Box's Branding either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Install Redux Framework either via the WordPress.org plugin directory, or by uploading the files to your server.
3. Activate both plugins.
4. That's it.  You're ready to go!
